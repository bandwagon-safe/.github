### 现有数据怎么备份

迁移第一步永远是先备份，不管目标平台多靠谱。方法很简单：

把重要数据打包压缩，通过 SCP 或 SFTP 拉到本地，或者上传到对象存储（比如阿里云 OSS）临时保存。数据库单独用 `mysqldump` 导出，不要图省事只备份整个磁盘镜像。

### 第五步：将数据从旧 VPS 迁移过来

# 用 SCP 传到新服务器

## 迁移后的常见问题处理

### IP 被封了怎么办

如果发现 IP 被封（可以在 KiwiVM 里用"Block List Check"功能检测），有两个选择：付费换 IP（目前价格约 8.79 美元一次），或者迁移机房换一个新 IP。注意：IP 已经被封的情况下不能直接迁移机房，必须先付费换 IP 才能操作。

### SSH 连不上怎么排查

连不上大致分几种情况：IP 被封、SSH 端口被封、sshd 服务挂掉、密码错误。可以先登录 KiwiVM 面板用在线终端（Root Shell）直接操作系统，绕过 SSH 限制进行排查。
